Privacy Policy

Tab Group Cleaner

Tab Group Cleaner is a Chrome extension that helps users find open tabs, close duplicates, group tabs by domain, and save or restore tab snapshots from a side panel.

Last updated: March 19, 2026 Works locally in Chrome

What the extension uses

Open tab titles, tab URLs, snapshot names, and saved preferences needed for core tab-management features.

Where data stays

Data is stored locally on the user’s device using Chrome storage. No remote code is used.

What is not done

No analytics, no advertising, no sale of user data, and no sharing of user data with third parties.

Information processed by the extension

To provide its tab management features, Tab Group Cleaner may process or store the following information locally on the user’s device:

  • Tab titles
  • Tab URLs
  • Snapshot names
  • Saved tab lists within snapshots
  • Theme and interface preferences

How this information is used

The extension uses this information only to provide its core functionality, including:

  • Filtering and finding tabs by title or domain
  • Detecting duplicate tabs
  • Switching to a selected tab
  • Grouping tabs by domain using Chrome tab groups
  • Saving and restoring snapshots of tab sessions
  • Storing user preferences such as theme selection

Data sharing and transfers

Tab Group Cleaner does not sell, transmit, or share user data with the developer or third parties.

No analytics, advertising, or tracking services are used by the extension.

Storage and retention

Snapshot data and preferences are stored locally in Chrome storage on the user’s device.

Users can remove stored data at any time by deleting snapshots, clearing extension storage, or uninstalling the extension.

Remote code

Tab Group Cleaner does not use remote-hosted executable code. All executable code is packaged with the extension.

Changes to this policy

If the extension’s privacy practices change in the future, this policy will be updated on this page.

Contact

For privacy questions about Tab Group Cleaner, use the contact email listed on the Chrome Web Store item page for this extension.
